# Project Documentation Folder

This folder contains all the necessary documents for your final year project presentation, report, and viva voce.

## Files included:
1. **[project_report.md](./project_report.md)** - Detailed project report with objectives, architecture, and technology stack.
2. **[viva_and_ppt.md](./viva_and_ppt.md)** - PPT slide outline and top Viva Voce questions with answers.
3. **[flowchart.md](./flowchart.md)** - Detailed Mermaid diagrams for System Architecture and Scan Logic.
4. **[code_samples.md](./code_samples.md)** - Core Python logic for ARP scanning and port auditing.
4. **[project_lifecycle.md](./project_lifecycle.md)** - Summary of the development process from start to finish.
5. **[walkthrough.md](./walkthrough.md)** - A step-by-step guide on how the project works and how to run it.

> [!TIP]
> Use these files to create your final project report and PPT presentation.
